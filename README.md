#  Shortcode Plugin (WordPress)

A simple and extendable **WordPress Shortcode Plugin** that allows administrators and developers to easily use and manage custom shortcodes inside pages, posts, and widgets.

This plugin demonstrates how to register and implement WordPress shortcodes using core hooks and functions.

---

##  Features

-  Custom shortcode registration
-  Easy to use in posts, pages, and widgets
-  Extendable for additional shortcode functions
-  Lightweight and performance-optimized
-  Works with any WordPress theme

---

##  Tech Stack

| Technology | Purpose |
|------------|---------|
| PHP        | Plugin logic and shortcode implementation |
| WordPress API | Integration with WordPress core |
| HTML       | Output markup for shortcodes |
| CSS        | Optional styling for shortcode outputs |
